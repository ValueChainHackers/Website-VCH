## Project Details
**Partner Organization:** Local Textile Cooperatives

**Start Date:** September 2024

**End Date:** January 2025

**Tags/Keywords:** textile recycling, circular economy, sustainability, web development, community engagement, waste reduction

**Project Summary:**
RE-USED tackles the growing problem of textile waste by creating an accessible web platform that connects people with recycling resources. The student team developed a Jekyll-based website featuring location finders, educational content about reuse practices, and community engagement tools. The project demonstrates how digital solutions can support circular economy initiatives at the local level.

**Key Outcomes:**
- Live website deployed via GitHub Pages
- Interactive recycling location finder
- Educational resources on sustainable textile practices
- Mobile-responsive design for community access
- Open source codebase for other communities to adopt
